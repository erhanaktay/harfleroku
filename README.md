# 🎮 Harf Canavarı

5 yaşındaki çocuklar için Türkçe harf öğrenme oyunu! Sesli, görselli ve eğlenceli.

## 🌟 Özellikler

- **3 Oyun Modu:**
  - 🔍 **Sesi Bul** — Verilen harfle başlayan kelimeyi bul
  - 🎯 **Harfi Eşleştir** — Aynı mantık, farklı soru tipi  
  - 👂 **Dinle & Seç** — Kelimeyi dinle ve tıkla
- 🔊 **Sesli anlatım** — Web Speech API ile Türkçe sesli okuma
- 🎉 **Animasyonlar** — Doğru/yanlış görsel geri bildirim
- ⭐ **Yıldız sistemi** — Kazanılan yıldızlar kaydedilir
- 📱 **Mobil uyumlu** — Tablet ve telefonda mükemmel çalışır

## 🚀 GitHub Pages'e Yükleme

1. **Yeni repo oluştur:**  
   GitHub'da `harf-canavari` adlı public bir repo aç.

2. **Dosyaları yükle:**
   ```
   index.html
   style.css
   game.js
   README.md
   ```

3. **GitHub Pages'i aktifleştir:**  
   `Settings` → `Pages` → `Source: main branch` → `/ (root)` → **Save**

4. **Oyunu oyna:**  
   `https://KULLANICI-ADIN.github.io/harf-canavari`
